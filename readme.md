# fe-newsstand

## HTML 레이아웃 설계

### 1. 기본 상단 영역 → header

뉴스스탠드 아이콘 & 제목이랑 오늘 날짜를 각각 div로 분리

뉴스스탠드 아이콘과 제목을 각각 span으로 분리

### 2. 최신 뉴스 자동롤링 영역 → section

뉴스 헤드라인 2개를 각각 div로

### 3. 언론사별 기사들을 확인하는 영역 → section

```html
<section>
  <div>
    <ul>
      <li></li>
      <li></li>
    </ul>
    <div>
      <span></span>
      <span></span>
    </div>
  </div>
  <div>
    <ul>
      <li></li>
      <li></li>
      ...
      <li></li>
    </ul>
  </div>
</section>
```
