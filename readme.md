# 네이버 뉴스스탠드

> 개발 기간: 23.07.05 ~ 23.07.28
> 개발 인원: 1주차 2명 (페어 프로그래밍), 2~4주차 1명

## 📌 프로젝트 소개

네이버 뉴스스탠드를 구현하는 과정을 통해 **바닐라 자바스크립트**의 기본과 심화 과정을 배운다.

##### 기능 요구 사항

- 롤링 애니메이션 구현
- 프로그래스바 구현
- 스토어 및 옵저버 패턴 구현
- 구독 및 해지 구현

- data fetching

- setInterval, setTimeout

## 📌 규칙

1. 폴더 구조 만들기

- [ ] src 폴더: js 로직 및 필요한 이미지를 관리하는 폴더
- [ ] commponent 폴더: js 컴포넌트 로직 관리
- [ ] header 폴더: 헤더 태그 컴포넌트 관리
- [ ] lastestNews 폴더: 최신 뉴스 컴포넌트 관리
- [ ] allNews 폴더: 전체 언론사 컴포넌트 관리
- [ ] common 폴더: 공통 컴포넌트 관리
- [ ] asset 폴더: 필요한 이미지 관리
- [ ] constants 폴더: 상수 관리
- [ ] utils 폴더: 유틸 관리

2. 필요한 asset 다운받기
