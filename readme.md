# 뉴스스탠드

## 네이밍 컨벤션

* 단어 사이를 언더바(_)로 구분한다.
* 시맨틱 태그 사용을 지향한다.

## 스타일

* reset.css와 constants.css 파일을 분리한다.

### reset.css

* 기본적인 스타일을 초기화한다.

### constants.css

* 공통적으로 사용되는 스타일을 css variable로 지정한다.

## 레이아웃 구조
